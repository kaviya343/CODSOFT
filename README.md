# TaskFlow - Smart Task Management System

TaskFlow is a Python-based To-Do List application developed using Tkinter and SQLite.

## Features
- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Priority Management
- SQLite Database Storage
- Task Statistics

## Technologies Used
- Python
- Tkinter
- SQLite3

## How to Run

1. Install Python.
2. Download the project files.
3. Open terminal or command prompt.
4. Run:

python taskflow.py
